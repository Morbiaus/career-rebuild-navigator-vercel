# Career Rebuild Navigator

AI-powered career rebuild and job placement navigator for professionals displaced by layoffs.

## Purpose

Career Rebuild Navigator is a working MVP that models an elite job placement agency using AI-agent style workflows. It helps a displaced professional enter basic career triggers, generate a transition profile, identify high-fit job lanes, and prepare a practical 30-day action plan.

## Current MVP Capabilities

- Candidate intake workflow
- AI Placement Team operating model
- Fit scoring against user career triggers
- Top job match recommendations from a built-in sample job market
- Resume positioning guidance
- Recruiter outreach message
- 30-day action plan
- Print/save as PDF capability

## Planned Enhancements

- Live job APIs
- AI-generated resume tailoring
- Saved application tracker
- User accounts
- Agent orchestration with governance checks

## Deployment

This is a static site and can be deployed directly on Vercel from GitHub.
