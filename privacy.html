Plain-English notice

Privacy and responsible use.

Career Rebuild Navigator is a prototype career-support tool. It helps users search for roles, evaluate fit, prepare application materials, and manage a job-search pipeline.

Do not paste sensitive personal identifiers. Users should remove Social Security numbers, birth dates, home addresses, personal account numbers, medical information, and other sensitive identifiers before using any page.

What is stored

  • Most tools run in the browser and generate text locally from information the user enters.
  • The Case Manager stores role-tracking data in the user’s browser local storage.
  • Local storage can be cleared by the user from the browser or by using the Case Manager clear button.
  • This prototype does not include user accounts or a central database.

What users control

  • Users decide what text to paste.
  • Users verify all job postings at the source.
  • Users approve every resume, cover letter, recruiter message, and application before use.
  • Users are responsible for ensuring every claim is truthful and evidence-supported.

What the tool does not do

  • It does not guarantee employment or interviews.
  • It does not make hiring decisions.
  • It does not represent employers or staffing agencies.
  • It does not provide legal, HR, financial, tax, or immigration advice.
  • It should not be used to misrepresent experience or credentials.

Recommended safe workflow

  • Use Market Search to find real postings.
  • Use Job Fit Review before applying.
  • Use CV Review to strengthen truthful positioning.
  • Use Application Package to draft materials.
  • Use Case Manager to track actions and follow-ups.
  • Apply only after human review and approval.
